## 背景
- 想查看[预览](https://preview.pro.ant.design/dashboard/analysis)的源代码
- 从仓库 clone 下来的是一个最基本的架子

## 解决
1. 下载 Ant Design Pro `git clone git@github.com:ant-design/ant-design-pro.git`
2. `yarn install`
3. `npm run fetch:blocks`
![](pro-blocks.png)
